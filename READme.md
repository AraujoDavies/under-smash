# UNDER SMASH

O projeto visa analisar se é válido entrar no mercado de under gols (lay over) ao fazer trading esportivo em jogos que tem uma expectativa de muitos gols, ou seja, começam a 2 parte do jogo com uma odd muito baixa.

## How to run ?

### pre-requisitos and env vars

    1. betfair account 

    2. betfair files in betfair-init folder: client-2048.crt and client-2048.key

Need to create a .env file and set the vars:

    BETFAIR_USER
    BETFAIR_PASSWORD

### To Run...

    git clone

    make up
